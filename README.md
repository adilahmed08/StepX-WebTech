# 👟 StepX – Sneaker E-Commerce Website

A responsive and modern **e-commerce web application for browsing and purchasing sneakers**, built using **HTML, CSS, JavaScript, and PHP**.

The project simulates a real sneaker store where users can browse products, filter items, manage a shopping cart, and place orders. The system also includes an **admin panel** for managing orders and customer information.

This project was developed collaboratively in a **3-person team** as part of a Web Technologies course.

# 🚀 Features

* 🛍️ **Product catalog** with sneaker images, descriptions, and prices
* 🔍 **Search and filtering** for browsing products
* 🧾 **Shopping cart system**

  * Add items to cart
  * Remove items from cart
  * Dynamic price updates
* 👤 **User authentication**

  * User registration
  * Login system
* 🧑‍💼 **Admin panel**

  * View customer information
  * Manage orders
* 📱 **Responsive design** for desktop and mobile devices
* 🎨 **Modern UI** inspired by real sneaker e-commerce platforms

# 💻 How to Run the Project

### 1️⃣ Clone the repository

```bash
git clone https://github.com/adilahmed08/StepX-WebTech.git
cd Stepx-WebTech.git
```

### 2️⃣ Install XAMPP

Download and install **XAMPP** if it is not already installed.

### 3️⃣ Setup project folder

Navigate to the XAMPP directory and open:

```
xampp/htdocs
```

Create a folder named:

```
step-x
```

Extract the project files into this folder.

---

### 4️⃣ Start the server

Open the **XAMPP Control Panel** and start:

* Apache
* MySQL

---

### 5️⃣ Create the database

Open the following link in your browser:

```
http://localhost/step-x/admin/create_databse.php
```

This will automatically create the required **database and tables**.

---

### 6️⃣ Launch the website

Open:

```
http://localhost/step-x/
```

You can now explore the website by:

* Registering a user account
* Logging in
* Browsing products
* Adding items to cart
* Placing orders
